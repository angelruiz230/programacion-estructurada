# Actividad: ciclos
## Descripción
en esta actividad vimos lo que son las estructuras de control ciclicas, while, do-while y for para lo que fue la creacion de un programa el cual te introduce un menu que te pide si quieres salir o quieres calcular un promedio, mediante la estructura de control do-while creamos un ciclo por eel cual se pueda pedir promedios de manera infinita hasta que se le aprete al boton de salir, el promedio se le pide al usuario la cantidad de calificaciones que va a promediar, una vez elija el numero de calificaciones, se mete en un ciclo for donde se hace la suma de todos las calificaciones y al final se dividen entre la cantidad de calificaciones que se usaron para el promedio, tambien se creo un programa que uso tambien el do-while para una calculadora de sumas y restas de dos numeros enteros o para analizar si un numero es par o impar. en este tema aprendimos a hacer menus con las estructuras de control ciclicas ademas de aprender a simplificar o hacer mas eficiente el codigo con el uso del for
## Temas aplicados

-condicionales
-estructuras de control ciclicas
