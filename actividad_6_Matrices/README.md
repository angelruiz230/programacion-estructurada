# Actividad: matrices
## Descripción
en esta actividad usamos matrices o arreglos bidimencionales para la creacion de un juego, inicializacion de tableros bidimencionales, y lectura de nombres con for anidados.
## Temas aplicados
-tipos de datos
-operadores
-condicionales
-estructuras de control ciclicas
-arreglos bidimencionales(matrices)
## Compilación
gcc matrices-juego-angel.c -o matrices-juego-angel
gcc matriz-nombres-angel.c -o matriz-nombres-angel


## Ejecución
./matrices-juego-angel
./matriz-nombres-angel
