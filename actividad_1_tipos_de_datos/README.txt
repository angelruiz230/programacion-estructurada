# Actividad: tipos de datos
## Descripción
Programa que muestra los tipos de datos primitivos el nombre, lugar en la memoria y el valor del la variable
## Temas aplicados
-tipos de datos enteros, flotantes y caracteres
## Compilación
gcc main.c -o tipos-de-datos-angel-ruiz
## Ejecución
./tipos-de-datos-angel-ruiz
