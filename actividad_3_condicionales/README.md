
# Actividad: condicionales
## Descripción
esta actividad consistio en 6 programas que usan las condicionales como su funcionamiento principal, se usaron los operadores tambien para la comprobacion de las condicionales, los codigos fueron acerca de validacion de numeros, usando if-else comprobamos si un numero era par o impar, sistemas de validacion de mayoria de edad, de si una persona era apta para una beca deacuerdo a si se cumplian ciertas condiciones esta practicas me ayudaron a dominar mejor el uso de este tipo de estructuras de control
## Temas aplicados
-tipos de datos
-operadores
-condicionales
