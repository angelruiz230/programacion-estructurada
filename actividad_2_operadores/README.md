
# Actividad: operadores
## Descripción
Programa operadores-angel-ruiz.c nos introduce a los operadores basicos utilizados en c, vemos como estos van afectando el valor de la variable imprimiendo el resultado de cada movimiento para entender mejor la logica dentro de estos operadores
el programa operadores-2-angel-ruiz.c nos pone 5 ejemplos pequeños de codigos donde se explica mejor el cambio de valor de las variables entender como funcionan logicamente dentro del programa y el 5to ejemplo es un ejercicio en el cual identificamos errores en el codigo y lo corregimos
## Temas aplicados
-tipos de datos enteros, flotantes y caracteres
-operadores
## Compilación
gcc operadores-angel-ruiz.c -o operadores-angel-ruiz
gcc operadores-2-angel-ruiz.c -o operadores-2-angel-ruiz

## Ejecución
./operadores-angel-ruiz
./operadores-2-angel-ruiz
