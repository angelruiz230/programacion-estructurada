# Actividad: arreglos
## Descripción
esta actividad trabajamos lo que son los arreglos para representar nombres, temperaturas listas de nombres y aprendimos a como recorrer estas listas de forma eficiente con for
## Temas aplicados
-tipos de datos
-operadores
-condicionales
-estructuras de control ciclicas
-arreglos
## Compilación
gcc arreglos-1-angel.c -o arreglos-1-angel
gcc arreglos-2-angel.c -o arreglos-2-angel
gcc arreglos-3-angel.c -o arreglos-3-angel
gcc arreglos-4-angel.c -o arreglos-4-angel
