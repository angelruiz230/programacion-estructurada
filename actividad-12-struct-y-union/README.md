# Actividad: struct y union
## Descripción
Se utiliza struct para agrupar múltiples variables bajo un mismo nombre y union para optimizar recursos compartiendo un mismo espacio de memoria entre diferentes variables
## Temas aplicados
-tipos de datos
-struct
-union
## Compilación
gcc struct-recorrido-angel.c -o struct-recorrido-angel
gcc struct-y-union-angel.c -o struct-y-union-angel

## Ejecución
./struct-recorrido-angel
./struct-y-union-angel
